Installing and Opening Vim

File Formats

Vim recognizes three file formats (Unix, DOS, and Mac) that determine line ending characters.

Installation

Visit Vim's Download Page and select the correct download link for your operating system. Vim comes pre-installed on Mac OSX as well as most Linux Distributions, so if you're using Mac or Linux, simply open a Terminal, type in "vim", hit enter and it should open for you! If it doesn't you can follow the steps on the Vim Download page to install it. Below is how to install Vim in Windows.

Windows

  1. After downloading the gvim74.exe file, navigate to the location you downloaded it to and double click the file.
  2. A window will pops up confirming that you want to install Vim, click "Yes" to begin the install
  3. The next screen prompts you with the license agreement. To continue insalling the program click "I Agree"
  4. After this the Installation Options show up. Here you can choose which features are installed. The default selections should be fine, so if you're new to Vim simply click "Next"
  5. The next screen lets you choose the location of the installation. Click "Install" to begin the installation
  6. Wait for the installtion to complete
  7. The next prompt confirms that the installation has completed successfully. Click Yes to view the Readme, or click No if you do not wish to view the Readme
  8. You are now ready to start using Vim!

Ubuntu Linux

Vim surprisingly doesn't come pre-installed in Ubuntu Linux anymore. The installation in Ubuntu is simple, simply type in the following command in a terminal.

sudo apt-get install vim

Opening Vim

Windows

If you stick with the default installation options on Windows, three Vim icons will appear on your computer desktop: gVim 7.4, gVim Easy 7.4, and gVim Read only 7.4. The only difference between standard and easy is that gVim Easy begins and locks in insert mode, whereas standard starts in normal mode and you can choose to toggle between modes. Learn more about modes.

Mac, Linux and Unix

In Mac, Linux and Unix based operating systems, open a terminal and type "vim" followed by a space and name of the file you want to edit or create. It's that easy! An example can be seen below.

vim foo.txt
