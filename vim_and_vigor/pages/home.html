Welcome!

Vim & Vigor is here to bring you an all-inclusive platform to begin using the Vim text editor and to learn more. By all-inclusive, we mean that any user with any level of experience in using Vim is welcome; beginners, intermediates, and experts. Why learn how to use a text editor? Aren't they simple to use? Not when they involve a number of commands, like Vim does, to make text editing more efficient. To reinforce your knowledge of using Vim, try some of our quizzes after completing the sections up to that point.

Beginners Tutorials

In beginner, we provide the basics for setting up and using Vim. From installation, navigating throughout the Vim workspace, Vim's various modes, and switching between modes. These steps are given step-by-step with supplementary screenshots for reference, along with links to read more about relevant steps.

Intermediate Tutorials

Intermediates are taken to the next level, with workflow, whitespace, and advanced search and replace methods. Given a bit more free-reign, these are given in a lenient manner, providing only the commands with a summary of what the command does and when you could perhaps benefit in using it.
