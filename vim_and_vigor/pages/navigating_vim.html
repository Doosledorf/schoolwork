Basic Vim Navigation

Command/Key Fuction Notes
h Move one space left For movement, any number typed before the movement command will execute for x number of words. For example, typing 3e will move you to the end of the word that is three words over.
i Move one space right
j Move one space down
k Moves one space up
e Move to the end of a word
w Move to the beginning of the next word
b Move backward to the beginning of a word
$ Move to the end of a line
0 Move to the beginning of a line
gg Move to the first line
G Move to the last line Typing a number before the command "G" will move you to the line that corresponds with that number. For example, typing 45G will move you to line 45.
) Move forward one sentence
( Move backward one sentence
} Move forward one paragraph
{ Move backward one paragraph
H Move to the top of the screen
M Move to the middle of the screen
L Move to the bottom of the screen
Move one page up Typing a number before a or command will move you that many pages up or down. For examples, typing 4 will move you four pages up.
Move one page down
'' Move to the line where the cursor was before the most recent jump This is two single quotes, not one set of double quotes.
`` Move to the cursor position before the most recent jump This is two back ticks, which can be found about the Tab key on most keyboard.
% Move to the corresponding item For example, this would move you from an open bracket to its matching close bracket.
cw Change word by deleting text and then entering insert mode Type a number between the "c" and the "w", or between the "c" and the "b" (when changing a word backwards) will change that many words. For example, typing c2d will change two words.
cb Change word backwards, similar to cw
dw Delete word and stay in normal mode The difference between cw and dw is that cw will delete the word(s) and then switch you to insert mode, while dw will delete the words(s) and keep you in normal mode.
u Undo Undo and redo will work multiple times rather than just once.
CTRL+r Redo
:w Save current file
:update Save current file The difference between :w and :update is that :w will save the file even if it has not been changed since the most recent save, while :update will only save the file if it has been changed since the most recent save.
:w! Save current file and override any problems found by Vim when saving the file
:w "filename" Save and name current file This words interchangeably with :saveas "filename".
:q Quit Vim
:wq Save current file and then quit Vim
:q! Quit Vim without saving any changes
:wa Save all open and changed files
:xa Save all open and changed files and quit Vim
:qa Close all open files and quit Vim unless there are unsaved changes
:qa! Close all open files and quit Vim regardless of any unsaved changes
f Move to the next specified character in the same line You must type a character after "f" (or after a F, t, T, ct, cf, dt, or df command). For example, typing fg will move you to the next g on the same line.
F Move to the previous specified character in the same line Like above, you must type a character after "F".
t Move to just before the next specified character in the same line
T Move to just after the next specified character in the same line
ct Change all text until the next specified character in the same line The ct and cf commands will both leave you in insert mode until you choose to return to normal mode.
cf Change all text until the next specified character in the same line, including the specified character
dt Delete all text until the next specified character in the same line
df Delete all text until the next specified character in the same line, including the specified character
